# Custom Backgroud
Стильный аддон для Яндекс Музыки, который позволяет менять глобальный фон приложения и фон блока «Моя волна» на любое изображение или видео.

# Особенности

- Двойная кастомизация: Раздельная установка фона для всего приложения и отдельно для раздела «Моя волна».

- Поддержка форматов: Полная поддержка статичных изображений (JPG, PNG, WEBP) и живых обоев (MP4, WEBM).

- Удобное управление: Кнопка управления интегрирована прямо в верхнюю панель рядом с системными кнопками.

- Производительность: Все медиафайлы сохраняются в локальную базу данных браузера, что обеспечивает мгновенную загрузку без лишних запросов к сети.

# Как пользоваться

1. В правом верхнем углу приложения появится кнопка "СМЕНА ФОНОВ".

2. При наведении кнопка плавно увеличится — нажмите на неё.

3. В выпадающем меню выберите нужный пункт:

    • Глобальный фон — изменит задний план всего приложения.

    • Фон Волны — изменит фон только внутри блока «Моя волна».

4. Выберите файл на вашем компьютере.

5. Страница автоматически перезагрузится, и новый фон будет применен.
